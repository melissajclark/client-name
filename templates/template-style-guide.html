Buttons and Forms

  • Buttons can use the Primary or Secondary Block Style.
  • Individual buttons can manually be set to have uppercase text using the Typography > Letter Case block setting.
  • All forms use Gravity Forms (except the Search form).
  • Button, text, and form styling is automatically updated according to the background colour of the section they are inside.
